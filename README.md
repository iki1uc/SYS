## SYS Symbols

Das SYS‑Symbol‑System definiert Host‑Typen:

- PQ = Laptop
- PX = Apple/Mobile
- PR = Desktop

Jedes Symbol besitzt:

- vector.EXP Routing (pipe3/6/9/12)
- QI / IQQ / OCTA Index
- RE‑Operationen (reCALL, reORGANIZE, REfinal)

Die Symbole werden über SYS/symbols.json geladen und im HH‑Panel angezeigt.
